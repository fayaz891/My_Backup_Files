DOM targeted main methods

we can target any DOM object by 3 main methods
1)By ID document.getElementById("id name")
2)by class name document.getElementsByClassName("Class name")
3)by tag name document.getElementsByTagName("tag name")

other methods


document (it target all document,all html targeted)
document.all (it targeted html all collections,all tags),if we want to target any spacific tag, then we can see its index no. in console and then mention that no. i.e document.all[3];
document.doumentElement
document.head (it targets all tags in head)
document.title (it targets title tag)
document.body (it targets all tags in body)
document.images (it target all images in a page)
document.anchors(it return all anchor tags)
document.links (it target all links in a page)document.links[5]; etc it target index no. 5 link
document.forms (it targets all forms in a page)
document.doctype (it return doctype of html,i.e html5 doctype, etc)
document.URL (it return URL of a website)
document.baseURI (it return all URL along with domain name)
document.domain (it return domain of a website i.e abc.com etc)
